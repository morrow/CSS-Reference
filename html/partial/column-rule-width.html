Summary

The column-rule-width CSS property lets you set the width of the rule drawn between columns in multi-column layouts.

  • Initial value : none
  • Applies to: all multi-column elements
  • Inherited : no
  • Media: visual
  • Computed value : absolute length; 0 if the column rule style is none or hidden

Syntax

column-rule-width:  <border-width>

Values

<border-width> 
See border-width.

Example

#header {column-rule-width: thick;}
