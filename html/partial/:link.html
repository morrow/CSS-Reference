Summary

The :link pseudo-class lets you select only links that have not yet been visited.

Examples

a:link {color:slaygrate;}
.external:link {background-color:lightblue;}

Specifications

  • CSS 2.1 Selectors #link
  • CSS 3 Selectors #links
