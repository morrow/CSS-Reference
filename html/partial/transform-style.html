Summary

The transform-style CSS property determines

  • Initial value : flat
  • Applies to : block and inline elements
  • Inherited : no
  • Media: visual
  • Computed value : same as specified value.

Syntax

transform-style: preserve-3d              or
transform-style: flat

where :

 

preserve-3d
