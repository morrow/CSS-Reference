Summary

The ::-moz-list-bullet pseudo-class is used to edit the bullet of a list element.

Syntax

li::-moz-list-bullet { style properties }

Examples

.liste li::-moz-list-bullet{ font-size:36px }

...where...

<ul>
<li>Number 1</li>
<li>Number 2</li>
<li>Number 3</li>
</ul>
<ul class="liste">
<li>Number 1</li>
<li>Number 2</li>
<li>Number 3</li>
</ul>

...result...
