Summary

-moz-tree-row is used to select rows and apply style to tree rows.

Associated Elements

  • treerow

Syntax

treechildren::-moz-tree-row { style properties }

Style Properties

  • background
  • border
  • margin
  • outline
  • padding
  • display
  • -moz-appearance

Examples

treechildren::-moz-tree-row( foo bar )
{
    margin: 2%;
}

...where...

  <treerow properties="foo">...</treerow>
