Summary

  • Initial value : none
  • Applies to: graphics and container elements
  • Inherited : no
  • Percentages: n/a
  • Media: visual
  • Computed value :

Syntax

filter: url(svg-url#element-id)

Examples

.target { filter: url(#c1); }

.anothertarget { filter: url(resources.svg#c1); }

Specifications
