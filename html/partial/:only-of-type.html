Introduced in Gecko 1.9.1

(Firefox 3.5 / Thunderbird 3.0 / SeaMonkey 2.0)

Summary

The :only-of-type pseudo-class represents any element that have no siblings of the given type.

Syntax

element:only-of-type { style properties }
