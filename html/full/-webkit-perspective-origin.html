CSS property: -webkit-perspective-origin

Description

Sets the origin of the -webkit-perspective property.

Sets the origin of the -webkit-perspective property described in "-webkit-perspective."

This property effectively sets the x and y position at which the viewer appears to be looking at the children of the element. The default value is 50% for both x and y coordinates.

Syntax

-webkit-perspective-origin: <posx> <posy>;

Values

<posx>

The x-origin as a percentage or value.

50% (by default)The default value is 50%.
centerSets the x or y origin to the center of the element’s border box. If this constant appears before left or right, specifies the y-origin. If it appears after top or bottom, specifies the x-origin. If appears alone, centers both the x and y origin.
leftSets the x-origin to the left side of the border box.
rightSets the x-origin to the right side of the border box.
<posy>

The y-origin as a percentage or value.

50% (by default)The default value is 50%.
centerSets the x or y origin to the center of the element’s border box. If this constant appears before left or right, specifies the y-origin. If it appears after top or bottom, specifies the x-origin. If appears alone, centers both the x and y origin.
topSets the y-origin to the top of the element’s border box.
bottomSets the y-origin to the bottom of the element’s border box.

Versions

  • iOS 2.0
